# 选择 YouTube 的关键指标

## 关键术语

| 术语 | 含义 |
| --- | --- |
| 产品经理（PM，Product Manager） | 负责产品目标、用户价值、优先级和落地执行。 |
| 北极星指标 | 捕捉产品核心价值、可粗略衡量成功的指标。 |
| 指标题框架（GAME） | 澄清问题、定义目标、梳理用户行为、选择指标、评估指标。本题主要使用目标、行为、指标、评估四步。 |
| 情感分析 | 用自然语言处理方法判断文本倾向是正面、负面还是中性。 |
| 观看时长 | 用户观看视频所花的时间。 |

## 题目

你是 YouTube Analytics 的产品经理（PM）。你会选择哪三个关键指标？为什么？

和很多产品经理题（PM questions）一样，这道题没有唯一正确答案。你需要给出合理答案，并为自己的观点辩护。

![YouTube 关键指标练习截图](../../assets/images/analytical-questions/exercises/youtube-key-metrics.png)

## 解题思路

这是一个产品管理指标题。通常不存在一个永远最适合优化的指标。产品经理常把最重要的指标称为北极星指标，因为它能捕捉产品核心价值，并粗略衡量产品是否成功。

这个答案使用指标题框架（GAME）：

| 步骤 | 在本题中的作用 |
| --- | --- |
| 目标 | 定义 YouTube 想达成什么。 |
| 行为 | 找出支持目标的用户行为。 |
| 指标 | 把用户行为转化为可衡量指标。 |
| 评估 | 讨论假阳性、假阴性和取舍。 |

真实面试中，最好先简单澄清范围。例如：“这里是衡量创作者使用的 YouTube Analytics，还是衡量 YouTube 作为观看平台的健康度？”本答案把题目理解为衡量 YouTube 平台的用户参与。

## 目标

YouTube 可以有多个合理目标：

- 用户留存。
- 用户参与。
- 更多视频上传时长。
- 通过 YouTube Premium 或广告变现。

本答案选择用户参与。YouTube 希望用户从娱乐性和教育性视频内容中获得有意义的价值。

收入通常不适合作为第一目标。只关注收入的公司可能忽视用户体验，而用户体验往往是更重要的长期健康指标。应该先提供值得用户投入时间的体验，再评估变现。

## 用户行为

能体现用户参与的行为包括：

- 给视频点赞。
- 观看视频。
- 评论视频。
- 点击视频缩略图。
- 将视频加入播放列表。
- 创建播放列表。
- 访问 YouTube 页面。

先分析行为，再进入指标。否则很容易跳过“产品目标如何连接到指标选择”的推理过程。

## 指标选择

本题中最关键的用户行为是：

- 观看视频。
- 给视频点赞。
- 评论视频。

为什么不选其他行为？

| 候选行为 | 为什么不是最核心 |
| --- | --- |
| 播放列表相关行为 | 有价值，但更偏小众功能，不足以代表平台整体健康度。 |
| 缩略图点击 | 能说明兴趣，但不能说明用户是否喜欢内容。 |
| 页面访问 | 能说明流量，但不能说明是否产生有意义的参与。 |

最终三个指标：

| 指标 | 衡量什么 |
| --- | --- |
| 每位用户平均视频观看时长 | 内容消费和观看深度。 |
| 每位用户平均点赞数 | 正向反馈和轻量认可。 |
| 每位用户平均评论数 | 主动参与和更强互动。 |

这些指标都以“每位用户平均值”定义，能更清楚地反映用户参与。

## 评估

这三个指标都有用，但都不完美。

| 指标 | 潜在问题 | 护栏或缓解方式 |
| --- | --- | --- |
| 每位用户平均评论数 | 评论可能代表沮丧、被冒犯或反感，不一定是正向参与。 | 用情感分析区分正面、中性和负面评论。 |
| 每位用户平均观看时长 | 极端观看时长可能代表沉迷、被动浏览或用户事后后悔。 | 追踪用户满意度、长期留存和负反馈。 |
| 每位用户平均点赞数 | 点赞可能受界面位置、创作者引导或推荐曝光影响。 | 搭配内容质量信号和满意度调查。 |

## 30 秒回答

我会把 YouTube 的目标定义为围绕娱乐性和教育性视频内容的健康用户参与。关键用户行为是观看视频、点赞和评论。因此我会选择三个指标：每位用户平均视频观看时长、每位用户平均点赞数、每位用户平均评论数。与此同时，我不会把它们视为完美指标，因为观看时长可能代表沉迷或被动浏览，评论也可能是负面情绪，所以我会搭配评论情感、用户满意度和长期留存作为护栏指标。
